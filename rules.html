游戏规则

一个把儿童糖分监控变成花园收集游戏的 PWA。吃得健康 → 花开得好 → 明天抽到更稀有的花。

🌱 目标

每次吃糖记一笔。糖量一多，今天的花就会蔫；糖量少，花开得漂亮。坚持好，第二天的抽卡概率变好，能收集到更多稀有花。

家人之间共用一个花园——爸爸妈妈在各自手机上看到同一朵花、同一块花田。

🎲 每日抽卡

每天首次打开 app 会弹出抽卡转盘，两阶段：

  1. 稀有度盘：5 个档次按 t² 缓动轮转，落在今天的档次
  2. 花种 carousel：该档次的花往左滑，cubic-bezier 2.8s 末端极慢落定

结果出现后可以 ✓ 保留 或 🔄 再抽一次（消耗一次机会）。

同一家庭所有设备同一天抽到一模一样的序列。妈妈抽第一次，爸爸也能在他手机上看到同样的结果。

🌸 花的品质

品质用 pacing 比例 算：糖量 ÷ (上限 × 时间进度)。也就是说，吃得早的糖比吃得晚的糖更伤花。

  • 早上 7 点吃 5g → pacing 很高（时间进度 0）→ 花立刻蔫
  • 一下午不吃 → 时间进度涨、糖量没涨 → pacing 下降 → 花慢慢回升
  • 午夜时分 pacing = 当日绝对 ratio，花进入最终态
品质Pacing 比例视觉
perfect<0.4最大最饱满
great0.4–0.7标准开放
ok0.7–0.9茎稍倾
okish0.9–1.0茎歪、花头斜
wilted1.0–1.3蔫花（去色、下垂）
dead≥1.3只剩枯枝

📊 动态概率

每次抽卡的稀有度概率 由昨天最终品质决定。吃得好 → 明天抽到高档花的机会越大。

🌻 种到花田

第二天打开 app 时，把昨天抽到的花种到花园里。48 个空位自由选，不用按日期顺序。

花田按行分大小——前排大、后排小，一起随风摆动。鼠标/手指滑过花田时，风向会跟着走。

点任何一朵已种的花可以看详情（花的品种、当天糖量、种下日期）。

👨‍👩‍👧 家庭共享

一个家庭最多几个人无限制（但每小时最多 300 次写）。爸爸、妈妈、外婆都可以加入同一个花园。

在设置页点 🔗 邀请家人 生成一个带 token 的链接，发给对方打开就能加入。记录谁吃了多少糖、种下哪朵花，所有人都能看。

邀请链接里的 token 是密码——只发给信任的家人，别贴公开群。

⏰ 错过一天

直接跳过。第二天打开 app 正常抽今天的花，不补昨天。已种的花也留在花田不动。

🧪 Debug 模式

URL 加 ?debug=1 开启调试模式：左上角 🐞 按钮弹出 Debug Panel，可以强制下次抽卡稀有度、重抽今日、给花田种指定稀有度的花、清空本机。

Debug 模式下同步自动关闭，改动仅本地。去掉 URL 参数即可恢复。
